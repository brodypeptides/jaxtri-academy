Sprint 6B

WooCommerce connector

Use this when you install the Jaxtri WordPress plugin on the WooCommerce store. The plugin tracks both links like ?ref=CODE and WooCommerce coupon codes used at checkout, then sends completed/cancelled/refunded order events here.
